"Mr. Stretch and the Stolen Fortune" Full Content Roadmap

This is the full content road map for Mr. Stretch. Please note that these are features as planned, and they are subject to change. Release dates will be added as they are planned.

Phase One: The Manor

This is the first of three phases, the development of Mr. Stretch is broken up into phases to improve the speed at which key features are developed.

This phase consists of all of the normal levels in the manor, and at its completion the first ending will be obtainable. This phase includes 100 levels, 5 optional bosses, and a final boss.

Version 0.2 "Ghost in the Plumbing"

This update introduces another area to Mr. Stretch: The Waterworks, a dangerous place where the player must race against a rising water level. This is planned to be released quarter 4, 2024.

Key features:

  • 20 Waterworks levels
  • Water valves and faucets
  • Warp pipes
  • Bubble wands
  • Goji boss fight

Version 0.3 "Green Thumb"

This update features the Conservatory area, where the player is chalanged to restore life to pretty, but mostly hazardous plants.

Key features:

  • 20 Conservatory levels
  • Growing vines/brambles
  • Gem flowers
  • New ghosts
  • Gaia boss fight

Version 0.4 "A Quiet Place"

This update introduces another area: The frozen library, where the sheer quantity of ghost has almost frozen the air itself.

Key features:

  • 20 Frozen Library levels
  • Ice slides
  • Ice sickles
  • A lot more ghosts
  • Charon boss fight

Version 0.5 "Out of Time"

This update features the Crumbling Clocktower. The clocktower is rife with incredible mechanical machines that prove dificult to navigate.

Key features:

  • Key and Lock Blocks
  • Automatic switch blocks
  • Cog blocks
  • Cannons and mirrors
  • Masque boss fight
  • Spirit king "final" boss fight
  • Ending #1

Phase Two: Secrets of The Manor

This Phase includes collectable journal pages and several secret areas that the player must find in order to beat the game the hard way. It includes a 6th world where the player can fight the actual final boss. The game will be almost completely done at this point, lacking some non-essential features.

Version 0.6 "Journal Update"

This update includes a journal interface overhaul and hidden collectable journal pages.

Key features:

  • Collectable journal pages
  • Journal overhaul
  • Hidden compartments for journal entries on several levels

Version 0.7 "Conspiracy Theory"

This update will add several hidden areas, a final 6th world, and the true final boss. Each world has 5 secret levels with even more unique obsticals. This update will complete the game, excluding non-core features.

Key features:

  • The Spirit King boss's second phase
  • 5 Grand Storage levels
  • Pushable blocks
  • 5 Ectoworks levels
  • Ectoplasm flood
  • 5 Seed Vault levels
  • Ancient seeds
  • 5 Archives levels
  • Hostile ghosts
  • 5 Electrical System levels
  • Electrical circutry
  • 20 Ancient Tomb levels
  • The Abyssal Horror boss fight

Phase Three: Finishing Touches

This phase will put some juice into the game, including atmospheric boss effects like Legion static, critters, dialog, and full cutscenes. This will include the full release of the game, at which point the game will be selling for full-price, and may possibly be ported to console.

Version 0.8 "Story Time"

Key Features

  • Dialog scenes
  • Dialog soundtrack
  • cutscenes

Version 0.9 "Better Bosses"

Key Features:

  • Improved bosses (bugfixes, visual improvements)
  • Harmless critters (Mice, Spiders, Frogs, Fishes, Bats)

Version 0.10 "Cheaty Face"

Key features:

  • Arcade mode
  • Speedrun mode
  • time attack

Version 1.0!!!

Key features:
